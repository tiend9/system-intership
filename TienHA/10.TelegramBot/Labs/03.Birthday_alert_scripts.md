# Viết script thông báo sinh nhật

## 1. File dữ liệu sinh nhật

Tạo file `birthdays.txt` với định dạng `<Tên> <MM-DD-YYYY>`:

```sql
Doan Tan 04-06
Thanh Binh 04-05
Anh Tien 06-04
Van Quyen 09-05
```

## 2. Script kiểm tra và thông báo sinh nhật

Tạo file `birthday_alert_bot.sh`

```bash
#!/bin/bash

# Bot và Channel information
BOT_TOKEN="8776668266:AAE6EMDjMBjxN9aFNehe7SFudMAD1_Yp2sY"
CHANNEL_ID="7989161444"  # Hoặc dùng ID: -100xxxxxxxxxx

# Today (DD-MM)
TODAY=$(date +%d-%m)

# Path to birthday data file (Hãy chắc chắn đường dẫn này đúng)
BIRTHDAY_FILE=""

# Kiểm tra xem file dữ liệu có tồn tại không
if [[ ! -f "$BIRTHDAY_FILE" ]]; then
  echo "Lỗi: Không tìm thấy file $BIRTHDAY_FILE"
  exit 1
fi

# Đọc từng dòng trong file
while IFS= read -r line; do
  # Bỏ qua dòng trống hoặc dòng có dấu # (comment)
  [[ -z "$line" || "$line" =~ ^# ]] && continue

  # Tách ngày sinh (từ cuối cùng) và Tên (các từ còn lại)
  BDAY=$(echo "$line" | awk '{print $NF}')
  NAME=$(echo "$line" | sed "s/ $BDAY//g")

  # Nếu trùng ngày sinh nhật với hôm nay
  if [[ "$BDAY" == "$TODAY" ]]; then
    
    # Gán tin nhắn vào biến một cách an toàn
    MESSAGE="*Birthday Alert!*
Today is *$NAME*'s birthday ($TODAY)"

    # Gửi API tới Telegram
    curl -s -X POST "https://api.telegram.org/bot$BOT_TOKEN/sendMessage" \
      -d chat_id="$CHANNEL_ID" \
      --data-urlencode text="$MESSAGE" \
      -d parse_mode="Markdown"
      
    echo "Đã gửi chúc mừng cho: $NAME"
  fi
done < "$BIRTHDAY_FILE"
```

## 3. Cấp quyền và thiết lập crontab

Cấp quyền thực thi:

```bash
sudo chmod +x birthday_alert_bot.sh
```

Mở cron:

```bash
crontab -e
```

Thêm nội dung dưới đây để chạy mỗi 8h sáng:

```cron
0 8 * * * /home/ubuntu/scripts/birthday_alert_bot.sh
```
