## 1. Hướng dẫn sử dụng

### 1.0 Tạo users

Chọn `Users`:

![WebVirtCloud](./images/WebVirtCloud_17.png)

Chọn `+`:

![WebVirtCloud](./images/WebVirtCloud_18.png)

Ta điền các thông tin cần thiết cho user này(**Lưu ý**: nên xem xét `user` kỹ càng cho phần `permission`)

![WebVirtCloud](./images/WebVirtCloud_19.png)

### 1.1 Tạo storage pool

Ở đây, ta tạo ra các Pool để lưu volumes VM và ISO để cài đặt OS cho VM:

- Vào tab `Computes` chọn `compute` bạn muốn tạo `pools`

![WebVirtCloud](./images/WebVirtCloud_20.png)

- Chọn `storage`:

![WebVirtCloud](./images/WebVirtCloud_21.png)

- Tạo 1 `pool` mới:

![WebVirtCloud](./images/WebVirtCloud_22.png)

=> Ở đây, ta thấy đã có sẵn 1 pool là `default`, ta sẽ lấy `pool` này làm nơi lưu images cho VMs

- Chọn đường dẫn lưu file-iso:

![WebVirtCloud](./images/WebVirtCloud_23.png)

- Sau khi tạo xong:

![WebVirtCloud](./images/WebVirtCloud_24.png)

### 1.2 Tạo network

### 1.3 Tạo VM

Để thực hiện được bước tạo VM user bạn phải là user có quyền `admin`

- Truy cập tab `Instances`, chọn `+`:

![WebVirtCloud](./images/WebVirtCloud_27.png)

- Chọn `compute`:

![WebVirtCloud](./images/WebVirtCloud_28.png)

- Chọn **kiến trúc**:

![WebVirtCloud](./images/WebVirtCloud_29.png)

- Chọn `Flavor`:

![WebVirtCloud](./images/WebVirtCloud_30.png)

- Điền các thông số cần thiết -> `Create`:

![WebVirtCloud](./images/WebVirtCloud_31.png)

- Chọn `setting`, sau đó chọn `Mount`:

![WebVirtCloud](./images/WebVirtCloud_32.png)

- Chọn `Power On`:

![WebVirtCloud](./images/WebVirtCloud_33.png)

- Chọn `Access`, sau đó chọn `Console`:

![WebVirtCloud](./images/WebVirtCloud_34.png)

- Tạo máy ảo thành công:

![WebVirtCloud](./images/WebVirtCloud_35.png)

### 1.4 Gán VM cho user thường

Chỉ có user `admin` mới có thể cấp VM cho user **thường** sử dụng. User thường chỉ có thể thao tác trên chính con VM được cấp chứ không phải là các `computes` chứa **VMs**:

- Chọn `tien9a`:

![WebVirtCloud](./images/WebVirtCloud_36.png)

- Chọn `+`:

![WebVirtCloud](./images/WebVirtCloud_37.png)

- Chọn VM thuộc compute và tùy chọn các quyền. Ở đây mình không cho phép user này xóa VM:

![WebVirtCloud](./images/WebVirtCloud_38.png)

- Login với user `tien9a`:

![WebVirtCloud](./images/WebVirtCloud_39.png)

- Màn hình quản trị của user `tien9a`:

![WebVirtCloud](./images/WebVirtCloud_40.png)

- User `tien9a` có quyền `resize`:

![WebVirtCloud](./images/WebVirtCloud_41.png)

- User `tien9a` không có quyền xóa VM:

![WebVirtCloud](./images/WebVirtCloud_42.png)

### 1.5 Snapshot

**NOTE:** Cần tắt VM trước khi `snapshot`

![WebVirtCloud](./images/WebVirtCloud_44.png)

Sau khi snapshot thành công ta sẽ thấy bản snapshot bên mục `Manager Snapshots`. Tại đây, ta cũng có thể thấy các bản Snapshot khác của VM:

![WebVirtCloud](./images/WebVirtCloud_45.png)

Quay lại bản snapshot nào đó, ta chọn bản snapshot rồi click biểu tượng như hình:

![WebVirtCloud](./images/WebVirtCloud_46.png)

Xóa bản Snapshot, ta chọn bản snapshot rồi click biểu tượng xóa như hình:

![WebVirtCloud](./images/WebVirtCloud_47.png)

Chọn `OK`:

![WebVirtCloud](./images/WebVirtCloud_48.png)
